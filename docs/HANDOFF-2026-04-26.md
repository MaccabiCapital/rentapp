## Critical rules that must NOT be broken

1. **Schema:** units use `unit_number` (not `label`); listings use `headline_rent` (not `monthly_rent`).
2. **Proof Check decision bar:** Approve / Request more info / Reject MUST stay all enabled regardless of risk band.
3. **Disparate-impact engine** must NEVER read name, email, phone, address, or any proxy. Bias-neutrality is a load-bearing acceptance test.
4. **All FairScreen rule packs are research-grade.** ATTORNEY-REVIEW-PENDING noted in seed + every rule file. Public marketing of FairScreen blocked until attorney sign-off (post-Phase-7 per founder Q answer; ~$2-5k).
